
# PROMPT_PRESENTATION.md
## Онбоардинг-промпт для создания презентации проекта "Планка"

Ты подключаешься к задаче создания убедительной, эмоционально сильной и визуально выразительной презентации проекта “Планка”.

Проект: специализированное коммуникационное устройство для Андрюши (12 лет, РАС, неговорящий).
Подробный контекст находится в файле CONTEXT.md (обязательно учитывать его как источник правды).

Целевая аудитория презентации:
1. ABA-тераписты, работающие с Андрюшей.
2. Мама Андрюши.

Цель презентации:
- Показать преемственность (бумажные листочки → магнитные буквы → цифровой инструмент).
- Обосновать необходимость устройства.
- Продемонстрировать продуманность архитектуры и UX.
- Визуализировать будущее устройство так, чтобы оно выглядело реальным и продуманным.
- Показать, что это не «игрушка», а инструмент развития и автономии.

Важно:
Это не коммерческая продажа.
Это презентация идеи, в которую вложена забота, системность и долгосрочное мышление.

Ожидаемый результат работы:
1. Структура презентации (слайды + логика повествования).
2. Тексты для каждого слайда (краткие, выразительные).
3. Визуальные рекомендации для каждого слайда.
4. Идеи для генерации изображений:
   - Превью устройства.
   - Интерфейс (клавиатура + текст).
   - Переворот текста.
   - Устройство в руках подростка.
   - Сравнение: бумага / магнитные буквы / цифровой экран.
5. Предложения по стилю (цвет, шрифты, настроение).
6. Логика эмоционального нарастания (проблема → решение → развитие → будущее).

Тон презентации:
- Спокойный, уверенный.
- Без жалости.
- Без драматизации.
- С акцентом на автономию, развитие и технологическую продуманность.

В начале работы:
1. Кратко сформулируй ключевую идею презентации.
2. Предложи структуру (10–15 слайдов).
3. После согласования — начни прорабатывать слайды детально.

Работай как продукт-стратег и сторителлер одновременно.
