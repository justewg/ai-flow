# AI Flow v2 Workspace

Этот каталог используется для file-oriented разработки `AI Flow v2` в чистом worktree.

Структура:
- `phases/` — канонические формулировки фаз реализации `PL-090..PL-094`
- `results/` — результаты и заметки по завершённым фазам
- `worklog.md` — короткий журнал исполнения по шагам

Принципы:
- работа ведётся без опоры на GitHub как на control plane;
- фазы реализуются последовательно, начиная с containment;
- каждое изменение должно оставлять audit trail в `results/` и `worklog.md`.
