# PL-090 — Phase A / Containment and SAFE mode

Цель:
- заморозить `v1` как legacy runtime;
- ввести `SAFE` / `EMERGENCY_STOP`;
- добавить грубые execution/budget counters;
- завести incident ledger.

Definition of done:
- новые expensive AI executions нельзя запустить без manual unlock;
- provider/quota failure переводит систему в stop-state;
- по задаче видны run count и last execution outcome.
