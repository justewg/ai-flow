# PL-095 — Integration 1 / runtime-v2 shadow bridge

Цель:
- подключить `runtime-v2` как отдельный shadow contour;
- materialize-ить v2 state из legacy runtime без переключения automation;
- держать все новые integration entrypoints с явным префиксом `runtime_v2`.

Definition of done:
- legacy state синхронизируется в отдельный persistent `runtime_v2/store`;
- есть snapshot entrypoint для просмотра v2 contour;
- legacy и v2 контуры не смешиваются по state files и wrapper names.
