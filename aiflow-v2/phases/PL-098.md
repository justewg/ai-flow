# PL-098 — Integration 4 / runtime-v2 primary source for selected contexts

Цель:
- сделать `runtime-v2` primary source хотя бы для selected lifecycle contexts;
- перестать полагаться только на legacy `daemon_active_*` / `daemon_review_*` файлы как на источник истины;
- сохранить миграцию обратимой и узкой.

Планируемый scope:
- derived primary contexts из `runtime-v2` для:
  - `executing`
  - `reviewing`
- отдельный reconcile-wrapper, который проецирует эти contexts обратно в legacy state files;
- `daemon_tick` и `watchdog_tick` начинают выравнивать legacy context из `runtime-v2` перед своей основной логикой.

Definition of done:
- selected `active/review` context в legacy runtime выравнивается из `runtime-v2`;
- end-to-end local smoke показывает, что `runtime-v2` может восстановить `daemon_active_*` и `daemon_review_*` без ручной записи этих файлов;
- cutover остаётся ограниченным и не затрагивает пока весь waiting-flow.
