# PL-102 — Integration 8 / make budget and circuit breaker authoritative from runtime-v2

Цель:
- перевести budget/quota/circuit-breaker decisions в explicit `runtime-v2` authority;
- убрать прямой shell-only stop path для provider quota и budget breach;
- сделать containment mode следствием `runtime-v2` state, а не отдельных несвязанных shell side effects.

Планируемый scope:
- orchestrator получает explicit `budget.breached` event;
- `runtime-v2` умеет выводить глобальный control mode из task budget states;
- shell получает отдельный sync-wrapper, который materialize-ит `SAFE` / `EMERGENCY_STOP` из v2 store;
- gate-path и provider quota path используют один и тот же authority channel.

Definition of done:
- `budget.breached` переводит task в paused/frozen budget stop внутри `runtime-v2`;
- `runtime_v2_sync_control_mode.sh` синхронизирует containment mode из `runtime-v2` store;
- `runtime_v2_gate.sh` после policy evaluation синхронизирует containment mode;
- provider quota path в `executor_run.sh` больше не делает shell-only emergency stop, а идёт через `runtime-v2`.
