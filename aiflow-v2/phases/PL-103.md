# PL-103 — Integration 9 / operator-grade runtime-v2 inspection surfaces

Цель:
- дать оператору компактную inspection surface поверх `runtime-v2`, не требующую чтения raw store и jsonl ledger-файлов;
- встроить `runtime-v2` summary в текущий `status_snapshot`, чтобы `/ops/status.json` автоматически получил v2-срез;
- сохранить inspection read-only и безопасной для `SAFE` mode.

Планируемый scope:
- отдельный inspection summary builder внутри `runtime-v2/src`;
- CLI и shell wrapper `runtime_v2_inspect`;
- врезка summary в `status_snapshot.sh`;
- runtime-v2 README и execution trail обновлены под новый operator surface.

Definition of done:
- `runtime_v2_inspect.sh` отдаёт compact JSON summary по control mode, task counts, primary contexts, incidents и execution summary;
- `status_snapshot.sh` включает section `runtime_v2`;
- summary покрыта unit-тестом и локальным shell smoke.
