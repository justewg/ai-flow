# PL-107 — Integration 13 / SAFE-mode target rollout kit and manual validation

Цель:
- подготовить reproducible SAFE rollout kit для целевой площадки;
- свести ручное участие к одному запуску и набору понятных артефактов;
- не включать automation и не выходить из manual validation mode.

Планируемый scope:
- shell kit для SAFE rollout;
- checklist с expected outcomes;
- локальная проверка kit в `--skip-git-sync` режиме.

Definition of done:
- есть `pl107_safe_rollout_kit.sh`;
- есть SAFE rollout checklist;
- kit проверена локально без сетевой синхронизации;
- если остаётся только запуск на целевой площадке, это явно помечено как operator action.
