Цель:
- подтвердить, что новый `micro` execution profile реально проходит end-to-end без full finalize;
- снять первую реальную telemetry на одном LLM call;
- убедиться, что `context_builder`, `llm_call_telemetry`, `canonical_diff` и deterministic metadata path реально materialize-ят execution artifacts.

Планируемый scope:
- добавить synthetic local canary harness без реального GitHub push/PR path;
- прогнать `micro` task через `executor_run` c fake `gh` и fake `codex`;
- сохранить `execution_profile.json`, `llm_calls.jsonl`, `execution_budget.json`, `check_results.json`, `canonical_diff.patch`.

Definition of done:
- synthetic canary классифицируется как `micro`;
- выполняется ровно один `implementation` call;
- telemetry и budget files materialize-ятся и содержат ненулевые token counters;
- verification checks проходят deterministic-path без участия LLM finalize.
