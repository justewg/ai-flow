# PL-124 — Target-aware micro context slices

Цель:
- убрать следующий blocker после live `micro` run `PL-124 / #549`;
- перестать подавать в `micro` prompt первые строки большого файла вместо релевантного фрагмента;
- перейти к anchor-based excerpt extraction из issue text для `context_builder`.

Планируемый scope:
- расширить `micro_profile_lib.sh`:
  - извлекать context anchors из issue title/body;
  - искать в target file совпадения по anchors;
  - собирать merged excerpt ranges вместо fixed top-of-file slice;
- обновить `context_builder.sh`, чтобы `micro_profile_file_context_json` получала полный issue text и строила target-aware file context.

Definition of done:
- большой target file в `micro` profile больше не режется как “первые 140 строк” по умолчанию;
- replicated issue body `PL-124` даёт compact excerpt вокруг `hero` block, а не шум из нерелевантных секций;
- excerpt остаётся deterministic и заметно меньше full-file context.
