# PL-130 — Executor handoff to normalized task spec

Цель:
- перевести executor handoff с raw issue body на `Standardized Task Spec`.

Планируемый scope:
- `context_builder` должен читать source/spec/intake artifacts вместо прямого `gh issue view`;
- `executor_build_prompt` должен строить prompt от normalized spec, а source definition оставлять только audit block;
- `executor_start` должен блокировать `human_needed | blocked` до старта executor;
- `micro` prompt больше не должен получать raw issue body как primary execution contract.

Definition of done:
- `context_builder` строит micro prompt из normalized spec;
- standard prompt тоже начинается с normalized spec;
- `human_needed | blocked` останавливаются до запуска executor;
- local smokes подтверждают новый handoff path.
