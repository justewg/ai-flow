# PL-091 Result Log

Статус: `done`

## Цель фазы
- поднять Mongo-backed state layer bootstrap;
- ввести минимальные сущности `Task / TaskState / Execution / Event`;
- обеспечить чтение runtime state без raw logs.

## Что делаем
- создаём `runtime-v2` package в shared toolkit;
- вводим store API и адаптеры `memory` / `mongo`;
- проверяем локально schema/store smoke tests без живой Mongo.

## Что реализовано
- Создан toolkit-local package `.flow/shared/runtime-v2`.
- Введены минимальные сущности и нормализация:
  - `Task`
  - `TaskState`
  - `Execution`
  - `Event`
- Добавлен единый `AiFlowV2StateStore`.
- Добавлен `memory adapter` для локальной разработки и тестов.
- Добавлен `mongo adapter` с env contract:
  - `AIFLOW_V2_MONGODB_URI`
  - `AIFLOW_V2_MONGODB_DB`
  - `AIFLOW_V2_MONGODB_COLLECTION_PREFIX`
- Добавлены базовые Mongo indexes для:
  - `tasks`
  - `task_states`
  - `executions`
  - `events`
- Реализован `getTaskBundle()` для чтения task/state/active execution/events без raw logs.
