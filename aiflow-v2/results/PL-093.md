## Что реализовано
- Добавлен execution control layer: `.flow/shared/runtime-v2/src/execution_policy.js`.
- Введён `buildExecutionDedupKey()` для identical expensive runs.
- Реализованы:
  - `acquireExecutionLease()`
  - `heartbeatExecutionLease()`
  - `releaseExecutionLease()`
  - `evaluateExecutionStaleness()`
  - `markExecutionStale()`
- Store/adapters расширены поиском execution по `dedupKey`.
- Duplicate expensive execution теперь даёт `noop` по dedup.
- Stale execution переводится в terminal `failed` без restart chain.
