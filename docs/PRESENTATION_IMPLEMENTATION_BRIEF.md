# PLANKA

## Web Narrative + PDF Export

### Implementation Brief v1.0

---

## 1. Общая цель

Создать web-first narrative-лендинг, который:

* демонстрирует эволюцию коммуникации
* обосновывает необходимость PLANKA
* показывает поведенческую механику
* визуализирует устройство
* остаётся минималистичным и взрослым

PDF генерируется через print-стили.

Это не маркетинговый сайт.
Это инженерный манифест инструмента.

---

## 2. Архитектура

Стек:

* Vanilla HTML
* CSS (grid + flex)
* Минимальный JS
* Без фреймворков
* Print.css для PDF

Каждая секция = full viewport height.

---

## 4. Медиаподдержка

В проекте подразумевается:

* Создание иллюстраций устройства (production-ready)
* Видео/Live-сцен для жеста
* Рендеры корпуса
* Минималистичный motion

Каждый медиаблок должен быть описан в коде как placeholder с comment-сценарием для дальнейшего заказа.

---

## 5. Дизайн-философия

* Тёмная тема
* CAPS
* Минимум цветов
* Матовые текстуры
* Никакого стартап-пафоса
* Никаких эмоциональных перегибов

---

## 6. Техническая дисциплина

* 1440px base width
* 12-column grid
* 8px spacing system
* Только opacity + translateY анимации
* Print-ready CSS
