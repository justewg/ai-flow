# PL-031 Проверка итеративного executor-flow

## Сценарий
- Коммит 1: выполнен первичный рабочий шаг по задаче.
- Вопрос: агент запросил подтверждение продолжения в Issue.
- Ответ: получен ответ пользователя `продолжай`.
- Коммит 2: выполнен завершающий рабочий шаг после `AGENT_RESUMED`.
- Финализация: запущен `task_finalize` для перевода задачи в `In Review`.

## Критерии проверки
- После вопроса включается waiting-state до пользовательского ответа.
- После ответа сценарий возобновляется без ручного сброса.
- Финализация публикует `CODEX_SIGNAL: FINAL_REVIEW`.
- PR переводится в `ready_for_review` при необходимости.
