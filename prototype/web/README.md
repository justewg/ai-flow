# PL-003 Prototype (Web)

Минимальный прототип экрана 50/50 для задачи `PL-003`.

## Что реализовано
- Верхняя зона: крупный вывод текста.
- Нижняя зона: детерминированная клавиатура RU/EN.
- Явное переключение языка (кнопка).
- Переключение темы light/dark (кнопка с луной/солнцем справа вверху).
- Стабильный split-layout 50/50.
- Попытка фиксации landscape через `screen.orientation.lock("landscape")`.
- Явный fallback в portrait с экраном-предупреждением и кнопкой повтора фиксации.
- Сохранение/восстановление текста и языка в `localStorage`.
- Без автокоррекции.
- Без скрытых состояний.

## Быстрый запуск
1. Открой `prototype/web/index.html` в браузере.
2. Переключи окно в landscape для проверки основного сценария.

## Ограничения текущей версии
- Это UI-прототип, не kiosk-режим.
- Нет TTS.
- Нет датчиков наклона.
- Поведение clear/backspace hold будет дорабатываться в задачах `PL-005+`.
